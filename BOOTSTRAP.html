History of bootstrap

Bootstrap is a free and open-source front-end web development framework that was created by Mark Otto and Jacob Thornton at Twitter in 2010. The framework was originally called Twitter Blueprint, and was designed as an internal tool to help developers at Twitter create consistent, responsive web interfaces.

The first version of Bootstrap, version 1.0, was released as an open-source project in August 2011. Bootstrap 1.0 included a set of CSS stylesheets and JavaScript plugins that could be used to create responsive layouts and common UI components, such as buttons, forms, and navigation menus.

Bootstrap quickly became popular among web developers due to its ease of use and versatility. The framework allowed developers to quickly create attractive and responsive web interfaces, without having to write custom CSS and JavaScript code from scratch.

In 2012, Bootstrap 2.0 was released, which included a number of new features and improvements, such as support for responsive typography, improved documentation, and a new customizer tool that allowed developers to customize the framework to their specific needs.

Bootstrap 3.0 was released in 2013, and included even more new features and improvements, such as support for mobile-first design, a new grid system, and a revamped set of CSS styles.

Bootstrap 4.0, which was released in 2018, was a major update to the framework, with a focus on modernizing the codebase and improving performance. Bootstrap 4.0 included a new Sass-based architecture, improved support for flexbox and other modern layout techniques, and a number of new components and utilities.

In addition to the core framework, Bootstrap also includes a number of add-ons and plugins that extend its functionality, such as themes, icon sets, and form validation plugins.

Today, Bootstrap is one of the most widely used front-end development frameworks, with a large and active community of developers and users. The framework is used by companies and organizations of all sizes, as well as by individual developers, to create responsive and attractive web interfaces quickly and efficiently.

Grid Basic

Bootstrap's grid system allows up to 12 columns across the page.

If you do not want to use all 12 columns individually, you can group the columns together to create wider columns

Bootstrap's grid system is responsive, and the columns will re-arrange automatically depending on the screen size.


colors

Bootstrap has some contextual classes that can be used to provide "meaning through colors".

The classes for text colors are: .text-muted, .text-primary, .text-success, .text-info, .text-warning, .text-danger, .text-secondary, .text-white, .text-dark, .text-body (default body color/often black) and .text-light
