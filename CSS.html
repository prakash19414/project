History of css

CSS (Cascading Style Sheets) was first introduced in 1996 by HÃ¥kon Wium Lie, a Norwegian web developer who worked at CERN, the European Organization for Nuclear Research. Lie developed CSS as a way to separate the presentation layer from the content layer of a web page, allowing developers to define the visual style of a website in a separate file or section.

The first version of CSS, CSS1, was published in 1996 as a recommendation by the World Wide Web Consortium (W3C), the international standards organization for the web. CSS1 included basic styling features such as font properties, colors, and backgrounds.

CSS2 was published as a recommendation in 1998 and included new features such as positioning and layout, as well as support for media-specific stylesheets. CSS2 also introduced the concept of cascading, where multiple style sheets can be applied to the same HTML document, with the most specific rules taking precedence over more general rules.

CSS3, the current version of CSS, is a modular standard that has been under development since 1999. It includes a wide range of new features, such as new selectors, media queries, and flexible box layouts. CSS3 also includes support for animations, transitions, and other advanced visual effects.
